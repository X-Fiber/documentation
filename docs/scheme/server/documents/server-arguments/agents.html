Агенти доступу

Агенти доступу призначені для представлення доступу до різноманітної функціональності, яке надає ядро обчислень. В перелік доступу входять - функціональність, яка реалізована в ядрі обчислень, доступ до різних модулів описаної бізнес-схеми, до функціональності інтеграційних рішень, а також до набору готових базових рішень, такі як CRUD чи інше.
